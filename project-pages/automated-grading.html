Predicting Student Essay Scores Using Document Embeddings and Neural Networks

View on GitHub

Description

This project was taken on for a Kaggle competition, where we were given a dataset of student written essays, and had to create a model which automatically scored each essay. By embedding each essay using the FlagEmbedding model, we were able to train a Neural Network that scored each document, on average, within roughly half a point of it's true score. This model was very competitive in a competition field of over 2000 teams.

Key Features

  • Regression model which calculated a numerical grade for student written essays
